# Flingoos Plugin for Claude Code

Connect Claude Code to [Flingoos](https://flingoos.com) — a context platform for workflows and knowledge bases.

## What's included

- **Flingoos MCP servers** — access workflows, knowledge bases, and integrations directly from Claude Code
- **Flingoos skill** — list, search, get, modify, and generate workflows and knowledge
- **Record skill** — start a workflow recording session from Claude Code with `/record`

## Installation

### Claude Code CLI (all plans)

```
/plugin marketplace add Diligent4/flingoos-plugin
/plugin install flingoos
```

### Claude Cowork (Max, Team, and Enterprise plans)

1. Open the **Plugins** panel in Cowork
2. Go to the **Personal** tab and click **Add marketplace**
3. Enter `Diligent4/flingoos-plugin` and click **Sync**
4. Install the Flingoos plugin from the marketplace list
